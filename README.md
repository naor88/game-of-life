## Setup The Project

1. Clone the repository
   ```bash
   git clone https://github.com/naor88/game-of-life.git
   cd game-of-life
   ```
2. Install the dependencies
   ```bash
   npm install
   ````

3. Build
   ```bash
   npm run build
   ```
4. Test
   ```bash
   npm run test
   ```

5. Run
   ```bash
   npm run start
   ```
